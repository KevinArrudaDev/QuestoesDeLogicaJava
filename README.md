# QuestoesDeLogicaJava
Este é repositório tem a função de armazenar os exercícios de lógica de programação voltado a Java.

São 100 questões totais, as quais são divididas em Sequências Básicas, Condições Básicas, Condições Compostas, Repetições Enquanto, Enquanto com Flag, Repetição com Faça Enquanto, Repetição com Para, Vetores, Procedimentos, Funçõe, totalizando 10 seções totais abordando variados temas.

O intuito é não somente concluir os exercícios, satisfazendo o que as questões pedem, mas também introduzir um tratamento de erros, com objetivo de prevenir a quebra dos programas.
